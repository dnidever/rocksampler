
RockSampler
===========

Sampler SSO orbits for tracklet data

Authors
-------

- **David Nidever** (Montana State University)
  
Installation
------------

To install the latest version::

    pip install git+https://github.com/dnidever/rocksampler

to install and use RockSampler.
