
SpaceRock
=========

Fit SSO orbits to tracklet information

Authors
-------

- **David Nidever** (Montana State University)
  
Installation
------------

To install the latest version::

    pip install git+https://github.com/dnidever/spacerock

to install and use SpaceRock.
